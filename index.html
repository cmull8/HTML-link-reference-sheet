HTML Hyperlink Reference

Hyperlink Tags

Tag Name Description
<a> hyperlink Link text wrapper. Also displays customized text or image.
<button onclick=url> hyperlink button Displays a button on the page that can redirect to another page.
<img> image hyperlink Image tags in <a> tags can act as hyperlinks.

Hyperlink Attributes

Attribute Name Description
href=url link Defines link destination.
target=_blank target Opens hyperlink on a new tab.
mailto:email@address.com email hyperlink Allows users to compose email to specified address.
title title Shows link title while cursor hovers over link.
